🧠 How Layout Preservation Works

For PDF, PowerPoint, Excel, and Word:

  1. 📸 Snapshot: The original document page/slide/sheet is rendered as a high-resolution image — capturing every font, color, table border, image, and layout detail.
  2. 🔍 Text Detection: Every text element's exact position (x, y), font size, width, and height is extracted from the original file.
  3. 🌐 Translation: Japanese text is translated to Nepali and English using Google Translate API.
  4. 🎨 Overlay: Translated text is placed at the exact same coordinates as the original Japanese text, with matching font sizes, using transparent-to-white backgrounds for clean readability.
  5. 📦 Export: The result is a new file that looks identical to the original — same paper size, same margins, same visual structure — just with translated text.

✅ Supported: Japanese → Nepali & English · PDF (all pages) · PowerPoint (all slides) · Excel (all sheets) · Word (all paragraphs)
⚠️ Complex formatting (3D effects, embedded videos, advanced charts) may have simplified rendering.
