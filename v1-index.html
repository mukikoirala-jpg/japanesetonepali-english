✨ New Features in v4.0:

  • 🎤 Speech-to-Text — Voice input for Japanese (Chrome/Edge only)
  • 🔊 Text-to-Speech — Listen to Nepali and English translations
  • 📖 Word Breakdown — See each word with furigana/ruby
  • ⚡ Auto-Translate — Translates as you type (with debounce)
  • 📝 Export as DOCX, TXT, HTML — More export options

Core Translation Engine:

  1. 📸 High-Res Snapshot: The original document page/slide/sheet is rendered as a high-resolution image — capturing every font, color, table border, image, and layout detail with pixel-perfect accuracy.
  2. 🔍 Intelligent Text Detection: Every text element's exact position (x, y), font size, width, and height is extracted from the original file structure.
  3. 🌐 Neural Translation: Japanese text is translated to Nepali and English using Google Translate API with context-aware precision and automatic retry on failure.
  4. 🎨 Smart Overlay: Translated text is placed at the exact same coordinates as the original Japanese text, with dynamic font scaling, using semi-transparent backgrounds for clean readability.
  5. 📦 Multi-Format Export: The result is a new PDF that looks identical to the original — same paper size, same margins, same visual structure — just with translated text.
✅ Supported: PDF · PPTX · DOCX · XLSX · XLS · CSV · TXT · HTML · RTF 🔄 Japanese → Nepali + English ⚡ Batch processing with progress tracking 📜 Translation History with export 🎤 Voice Input/Output

⚠️ Complex formatting (3D effects, embedded videos, advanced charts) may have simplified rendering.
